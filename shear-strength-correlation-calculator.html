Global Parameters
N₆₀ = N × CE × CR CE = ER / 60 (N1)60 = N₆₀ × CN CN = 0.77·log(20 / σ′v,tsf)  [coarse] CN = 1.0  [fine] σ′v cumulative; γ → γbuoy below GWT
Two correlations, one per soil class: coarse-grained (granular) samples return a friction angle φ from Peck, Hanson & Thornburn (1974); fine-grained (cohesive) samples return an undrained shear strength Su = (N1)60 / 7.5 ksf. Per the source spreadsheet convention, the overburden correction CN is set to 1.0 for fine-grained soils, so (N1)60 = N₆₀ for those samples.
